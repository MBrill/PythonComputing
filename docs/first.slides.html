Unsere erste Folie¶

Wie in jedem anderen Notebook können wir Markdown und Python Cells einsetzen. In einer Präsentation gibt es weitere Typen wie

  • Slide
  • Sub-Slide
  • Skip
  • Notes

Wir können für die Cells sichtbar machen welche Funktion sie haben mit View -> Cell Toolbar -> Slideshow.
